## Лабораторная работа №1: Многослойный персептрон

### Задание

Разработка и обучение многослойного персептрона для задачи бинарной классификации грибов на съедобные и ядовитые.

### Набор данных

- Источник: UCI Machine Learning Repository
- Ссылка: https://archive.ics.uci.edu/ml/datasets/Mushroom
- Описание: Датасет содержит описания характеристик грибов с соответствующей классификацией (съедобный/ядовитый)

### Требования к решению

#### Структура сети

- Входной слой: соответствует количеству признаков
- Скрытые слои: 2 полносвязных слоя
- Выходной слой: 2 нейрона (бинарная классификация)

#### Параметры обучения

- Функция активации: сигмоида
- Метод оптимизации: градиентный спуск
- Использование сторонних оптимизаторов запрещено

#### Метрики оценки

Необходимо рассчитать следующие метрики качества:

- Accuracy
- Precision
- Recall
- F1-score

#### Визуализация результатов

Построить ROC-кривую для оценки качества классификации.
